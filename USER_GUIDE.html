Ceph Doc KB

User Guide — Search Ceph documentation, code examples, and how-to guides from your AI agent

1. What is Ceph Doc KB?

Ceph Doc KB is an MCP server that provides semantic search across the official Ceph documentation. It indexes RST docs by component (rados, rbd, rgw, cephfs, cephadm) into per-component vector indices for fast, targeted search.

  • Search documentation with natural language queries
  • Find code examples and configuration snippets
  • Look up docs for any Ceph command instantly
  • Browse by component and topic

Two-tier search: BM25 for exact terms, fastembed for conceptual queries. Component-scoped for speed and accuracy.

2. Connect Your Agent

Option A: Cursor (stdio)

{
  "mcpServers": {
    "ceph-doc-kb": {
      "command": "python3",
      "args": ["-m", "ceph_doc_kb.server.mcp_server"],
      "cwd": "/path/to/ceph-doc-kb"
    }
  }
}

Option B: HTTP/SSE (Bob, Claude Desktop, Continue, Cline)

python -m ceph_doc_kb.server.mcp_server --transport sse --host 0.0.0.0 --port 8082
{
  "mcpServers": {
    "ceph-doc-kb": {
      "url": "http://localhost:8082/sse",
      "transport": "sse"
    }
  }
}

6. Troubleshooting

  • No results for search — Try a different component with list_components, or search without a component filter.
  • Agent doesn't see the server — Restart the IDE after editing mcp.json.
  • SSE connection refused — Verify: curl http://localhost:8082/health.
